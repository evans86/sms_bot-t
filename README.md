<h1>Install Project</h1>
<hr>
git clone<br> 
composer install<br>
php artisan migrate<br>
<b>Применим сиды:</b> <br>
php artisan db:seed <br>

<hr>
$npm install<br>
<i>development окружение</i><br>
$npm run dev <br>
<i>production окружение</i><br>
$npm run prod
